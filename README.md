# Hyderabad Flood Risk Map

This mini-project analyzes Hyderabad's elevation data to identify areas at risk of flooding using QGIS. 

## Objective
To visualize low-lying flood-prone areas (<550 m) in Hyderabad for disaster management and urban planning.

## Tools Used
- **QGIS** (Raster calculations, visualization)
- DEM data (Digital Elevation Model)
